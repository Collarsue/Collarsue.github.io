如何客製化自己的 p10k 色彩配置

每次安裝 oh-my-zsh 跟 p10k 時無論是使用 Monokai 還是 OneDark，都覺得 p10k 的色彩配置一成不變嗎？

其實這可以透過修改~/.p10k.zsh來客製化自己的 p10k

前期準備

  1. 使用 sudo apt install zsh 或是 sudo brew install zsh 來安裝 zsh
  2. 安裝oh-my-zsh
  3. 安裝 powerlevel10k(p10k)

使用指令瀏覽終端機目前的 255 色票

輸入以上的指令會得到下圖的一大塊色票，所有你可以用的顏色都會在這裡

打開你的.p10k.zsh開始客製化吧

vim ~/.p10k.zsh

鍵入/OS_ICON搜尋一下

WOW，這時候看到 106 行跟 107 行的程式碼尾段分別寫著數字

POWERLEVEL9K_OS_ICON_FOREGROUND=208
POWERLEVEL9K_OS_ICON_BACKGROUND=238

這其實就是剛剛看到的色票數字，只要把改成自己喜歡的色票號就可以改變顏色

那還有沒有其他能改的？當然有，都寫在.p10k.zsh裡

有個小訣竅，只要透過 BACKGROUND 或是 FOREGROUND 這兩個關鍵字去找，大概都能找到對應的色票，像是：

# 資料夾路徑顏色
POWERLEVEL9K_DIR_BACKGROUND=
POWERLEVEL9K_DIR_FOREGROUND=

# Git區塊顏色
POWERLEVEL9K_VCS_CLEAN_BACKGROUND=
POWERLEVEL9K_VCS_MODIFIED_BACKGROUND=
POWERLEVEL9K_VCS_UNTRACKED_BACKGROUND=
POWERLEVEL9K_VCS_CONFLICTED_BACKGROUND=
POWERLEVEL9K_VCS_LOADING_BACKGROUND=

# 狀態OK顏色
POWERLEVEL9K_STATUS_OK_FOREGROUND=
POWERLEVEL9K_STATUS_OK_BACKGROUND=

可以調整顏色的地方太多，就不全部列出來了

結語：.p10k.zsh內還有很多的設定可以調整，甚至可以加入一堆 Emoji 來代替一些icon，不過此篇主要是講述如何修改顏色，就不特地贅述了。
